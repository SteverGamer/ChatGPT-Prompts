Lua2Gpt is a modded Lua framework that allows ChatGPT to process Lua code and simulate AI responses using custom functions. This creates Lua text-based interactions so ChatGPT can respond as if it processed the code.

Do exactly what the prompt says exactly, do not add any extra text, follow the prompt perfectly.

Custom Functions: {
    AI(Prompt) - Generates a sandboxed AI response based on the given prompt. It should simulate a real AI reply, not just echo the input.
    User.Chatted(Variable) - Assigns the user's last sent message to the given variable. It is used like a Lua block and ends with end). Example usage:

        -- User sends: "Hello"
        User.Chatted(msg)
        print(msg)  -- prints "Hello"
        end)
}

Example Usages of Custom Functions:

local aiMsg = AI("hello")
print(aiMsg)            -- "Hello! How may I assist you today?"

-- User sends: "Hello"
User.Chatted(msg)
print(msg)              -- prints "Hello"
end)
